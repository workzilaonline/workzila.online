How it works

  1. User clicks "Apply" and fills the Google Form (embedded below).
  2. Form responses go to your Google Form / Google Sheet (you already created it).
  3. Set up Google Sheets notification or Apps Script to forward submission details to your phone (instructions below).
Quick setup: get form->phone notifications
  1. Open your Google Form > Responses > Click the Google Sheets icon to send responses to a sheet.
  2. In that Google Sheet: Extensions > Apps Script — paste the sample script (below), add your Twilio or SMS gateway credentials, and deploy as Web App or run onSubmit trigger.
  3. Or use services like Zapier/Make/IFTTT to forward a new row to SMS.
    Note: PhonePe cannot directly receive form data — use SMS gateway or email-to-SMS instead.
// Sample Google Apps Script (send SMS via Twilio)
function sendSmsOnFormSubmit(e){
  // set these values:
  var accountSid = 'TWILIO_ACCOUNT_SID';
  var authToken = 'TWILIO_AUTH_TOKEN';
  var fromNumber = '+1234567890'; // your Twilio number
  var toNumber = '+91XXXXXXXXXX'; // your phone

  var formValues = e.values; // full row values
  var message = 'New lead from WorkZila: ' + formValues.join(' | ');

  var payload = {
    'To': toNumber,
    'From': fromNumber,
    'Body': message
  };

  var options = {
    'method':'post',
    'payload': payload,
    'headers': {
      'Authorization': 'Basic ' + Utilities.base64Encode(accountSid + ':' + authToken)
    }
  };

  UrlFetchApp.fetch('https://api.twilio.com/2010-04-01/Accounts/' + accountSid + '/Messages.json', options);
}

// Then set a trigger: When form is submitted -> sendSmsOnFormSubmit
